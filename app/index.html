Трубы из полиэтилена для газопроводов ГОСТ Р 50838-2009

Трубы из полиэтилена (ПЭ) более 50 лет применяются во всём мире

Газопроводные системы из полиэтилена применяются для сетей газораспределения, в т. ч. высокого давления I категории. Трубы ПЭ предназначены для транспортировки горючих газов по ГОСТ 5542, прошли экспертизу промышленной безопасности в лицензированной Ростехнадзором экспертной организации, имеют сертификат соответствия EN 1555-2:2010

Гарантия

Экологичность

Пригодна для питьевой воды
